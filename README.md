# F1 Experience – Misión Clasificada

Una experiencia web interactiva inspirada en la Fórmula 1 diseñada para revelar un regalo de forma inmersiva.

El usuario comienza una simulación de conducción con secuencia de semáforos, telemetría en tiempo real y análisis dinámico del piloto. Al completar la carrera, recibe un informe personalizado que da paso a la revelación final: una sesión **Head To Head** en F1 Arcade Madrid.

## Características

* Pantalla de introducción estilo briefing clasificado.
* Secuencia de salida con semáforos y efectos de sonido.
* Simulación de carrera con:

  * Velocidad dinámica.
  * Indicador de marchas.
  * Tacómetro animado.
  * Análisis del piloto en tiempo real.
  * Estado del vehículo (ERS, DRS, neumáticos y temperatura).
* Animación de línea de meta.
* Informe final con evaluación del rendimiento.
* Revelación del regalo mediante una tarjeta interactiva.

## Ejecución local

Abre el archivo `index.html` directamente en el navegador o utiliza un servidor local.

Por ejemplo, con Visual Studio Code y la extensión Live Server:

1. Instala la extensión.
2. Haz clic derecho sobre `index.html`.
3. Selecciona "Open with Live Server".
